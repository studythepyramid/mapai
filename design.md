There's going to be layers of maps, 
the first layer is like google map or OSM map, 
it serves as base map.  
The 2nd would be virtual map on the real, 
user could write text, leave voice messages on the 2nd map.
User could also post pictures and videos on the 2nd, 3th layers of maps, etc.
User could build virtual houses, or other buildings without 
conflicts with the properties occupations in the base map.

user should be able to mark dangerous things on the map,
such as aggressive dogs, poisonous snakes, thieves, 
gun shots, explosives, etc. 

Build a list of second-hand items, like craiglist,
the categories and items should be adhered to the map.

User could offer messages of job seeking and recruitment,
as Jobs and Gigs on craiglist.

User data should be kept on user's devices, 
android or iphone, unless they pay for online storages,
we need DB for it.

We're going to carry out the development on my Ubuntu 26.04, Wayland.
And we're going to start it with multiple programming languages, 
python is for fast prototyping, 
Rust and others serve for other purposes.
edit the .gitignore for the multi-lang tasks.


user could be able to team up and detect the dangerous objects on the map,
for example cheatings and kidnappings.

user should be able to turn to emergent status,
when all possible information transferred to DB,
such as positional information, text, audio/voice, camera captures.

AI help on the maps, it's LLM models.
the model could be local and online as well.
